# 排序问题

## 排序的几个基本概念

内排序：在对待排序数据存放在内存中进行的排序过程。
外排序：待排数据量太大，无法一次性将所有待排序数据放入内存中，在排序过程中需要对磁盘等外部储存器进行访问。
比较排序：排序过程中需要对数据关键字进行比较。
非比较排序：排序过程中不需要对数据关键字进行比较。
排序算法的稳定性：在每一次单趟排序后，相同关键字的相对顺序不变。
